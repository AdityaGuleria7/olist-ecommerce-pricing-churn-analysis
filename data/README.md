# Data

The raw dataset files are not included in this repository due to size.

## Download Instructions

1. Go to: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
2. Click **Download** (requires a free Kaggle account)
3. Extract the ZIP — you will get 9 CSV files
4. Upload all 9 CSVs to your BigQuery project as described in the main README

## Files needed

| File | Rows | Description |
|---|---|---|
| olist_orders_dataset.csv | 99,441 | All orders with status and timestamps |
| olist_order_items_dataset.csv | 112,650 | Products per order with price |
| olist_customers_dataset.csv | 99,441 | Customer location data |
| olist_products_dataset.csv | 32,951 | Product details and category |
| olist_order_payments_dataset.csv | 103,886 | Payment method and value |
| olist_order_reviews_dataset.csv | 100,000 | Customer review scores |
| olist_sellers_dataset.csv | 3,095 | Seller location data |
| olist_geolocation_dataset.csv | 1,000,163 | Brazilian zip code coordinates |
| product_category_name_translation.csv | 71 | Portuguese to English category names |
